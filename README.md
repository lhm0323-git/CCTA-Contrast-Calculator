# CCTA 顯影劑劑量與注射流速精準計算工具 (Web-based Tool)

本專案為一個專門為高階心血管健檢中心設計的 **CCTA (冠狀動脈電腦斷層造影)** 顯影劑與生理鹽水精密注射劑量及流速計算系統。基於 **Somatom Force** 雙源斷層掃描儀的 80 kVp 低劑量造影協定，能有效避免造影劑腎病變 (CIN) 並且快速算出最適劑量。

---

## 核心設計特色

### 1. 實測總體重基準與足量對比 (Optimal Contrast)
- **以總體重 (Total Weight) 作為劑量計算的預設核心基準**，避免了去脂體重 (LBM) 所致初始劑量過低、導致 CCTA 對比度可能不足的缺點。
- 預設帶入推薦的標準對比參數，確保在預設狀態下的碘流速 (IDR) 即處於最佳對比範圍（**1.4 至 1.8 g I/s** 之間）。

### 2. 臨床級離散設定級距 (Clinical Rounding)
為了貼合放射師的高壓注射器操作直覺，系統對計算結果進行了人性化離散分級處理：
- **建議顯影劑量 (Volume)：自動以每 5 cc 為一個級距進行四捨五入** (例: 49.6 cc 自動轉換為 50 cc)，並設有最小安全劑量限制 (20 cc)。
- **建議注射流速 (Flow Rate)：自動以每 0.5 cc/s 為一個級距進行四捨五入** (例: 4.17 cc/s 自動轉換為 4.0 cc/s)。
- **實際注射秒數：** 根據四捨五入後的「建議劑量 / 建議流速」重新精準計算出時間 (例: `50 cc / 4.0 cc/s = 12.5 秒`)，最符合臨床上高壓注射器的設定與運作機制。
- **實際碘流速 (IDR)：** 依四捨五入後的流速和含碘濃度重新計算，保證數據與高壓注射器運作完全一致。

### 3. 心率自動適應注射協定 (Heart Rate Adaptation)
針對受試者心跳過快的情形，系統會自動調節注射流速與注射秒數：
- **心跳 < 65 bpm：** 採用常規低心率協定，設定為 **12.0 秒** 注射時間。
- **心跳 65–75 bpm：** 判定心跳偏快，為維持造影前沿對比濃度，自動將注射時間縮短至 **11.0 秒**，進而提高流速 (mL/s)。
- **心跳 > 75 bpm：** 判定心跳過快，自動縮短注射時間至 **10.0 秒**，以強化高心率下的團注 (Bolus) 對比劑濃度。
*(放射師亦可在進階摺疊面板中，手動拖曳滑動條來覆蓋心率的自動設定。)*

### 4. 直接雙擊執行 (開箱即用)

## 核心臨床計算演算法與公式

### 1. 身體質量指數 (BMI)
$$BMI = \frac{\text{體重 (kg)}}{\text{身高 (m)}^2}$$

### 2. 目標總碘量 (Total Iodine Load) 與電壓校正
碘劑量會根據選用的掃描電壓 (kVp) 進行校正。
- **電壓校正係數 (Voltage Multiplier)：**
  - **70 kVp:** 0.8
  - **80 kVp (預設):** 1.0
  - **100 kVp:** 1.2
  - **120 kVp:** 1.4
- **總體重基準目標總碘量 (mg)：**
  $$\text{總碘量} = \text{體重} \times \text{總體重碘劑量係數 (預設 270 mg I/kg)} \times \text{電壓校正係數}$$

### 3. 顯影劑注射建議總體積 (Contrast Volume)
$$\text{建議體積 (cc)} = \text{Round}_{5} \left( \frac{\text{目標總碘量 (mg)}}{\text{顯影劑含碘濃度 (mg I/mL)}} \right)$$
*(四捨五入以 5 cc 為一級距。選用 Xenetix 350 / Iopamiro 370)*

### 4. 建議注射流速 (Flow Rate)
$$\text{建議流速 (cc/s)} = \text{Round}_{0.5} \left( \frac{\text{建議體積 (cc)}}{\text{注射秒數 (秒)}} \right)$$
*(四捨五入以 0.5 cc/s 為一級距)*

## 臨床安全防護鎖定與覆核機制 (eGFR Gates)
展開「臨床安全卡控 (eGFR)」面板，可進行患者安全卡控：
- **eGFR ≧ 45 mL/min/1.73m² (綠色安全區)：** 正常計算。
- **eGFR 30–44 mL/min/1.73m² (黃色警示區)：** 警示橫幅，提醒加強水份灌注。
- **eGFR < 30 mL/min/1.73m² (紅色鎖定區)：** 劑量計算自動鎖定遮罩。
  - *解鎖機制：* 需由專科醫師在安全卡牌中鍵入「授權醫師姓名」與「覆核解鎖原因」並送出，即可解除鎖定，記錄將存於覆核日誌中。

## 快速啟動與預覽

### 方法一：直接雙擊執行
在檔案總管中雙擊 `index.html` 即可於 Chrome 或 Edge 中運行，完美解決 CORS 跨網域阻擋問題。

### 方法二：啟動輕量本地伺服器
在專案根目錄下執行以下指令：
```bash
npx -y http-server "f:/github/ccta contrast" -p 8080
```
啟動後在瀏覽器輸入 `http://localhost:8080` 即可預覽。
